## 方法一

第一種方法，展示如何印出stdout

使用簡單例預設顯示在右側欄位

{% sample lang="js" %}
如何用JavaScript印出`stdout`
```js
console.log('My first method');
```

{% sample lang="ccc" %}
如何用C/C++印出`stdout`
```c
printf("My first method");
```
or another way, 
```
cout << "My first method";
```

{% common %}
無論使用何種方法結果都一樣：

```bash
$ My first method
```
