## 方法一

第一種方法，展示如何印出stdout

{% sample lang="js" %}
Here is how to print a message to `stdout` using JavaScript.

```js
console.log('My first method');
```

{% sample lang="c/c++" %}
Here is how to print a message to `stdout` using C/C++.

```c
printf("My first method");
```
or another way, 
```
cout << "My first method";
```

{% common %}
無論使用何種方法結果都一樣：

```bash
$ My first method
```
